# 模型视图控件

控件（control）是一种模型视图的基本用户界面元素，例如选择按钮或文本框等，其显示的信息排列可由用户改变。在模型视图中，控件为给定数据的直接操作（direct manipulation）提供**单独的互动点**，是一种基本的可视构件块，控制着Matrix 程序组件处理的所有数据以及关于这些数据的交互操作。
